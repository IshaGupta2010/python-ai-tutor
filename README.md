# AI Python Tutor

Welcome to the AI Python Tutor! This interactive tutor uses LangChain and Ollama's Mistral model to help you learn Python through examples and coding challenges. It runs on your local machine and supports memory retention and feedback.

## üìö Features

- **Interactive Python Tutor**: Chat with the tutor and get Python concepts explained.
- **Memory Retention**: Remembers previously learned concepts.
- **Coding Challenges**: Solve small coding challenges and receive feedback.
- **Offline-Ready**: No internet required once set up.

## üõ†Ô∏è Tech Stack

- **Python**: The core language for running the tutor.
- **LangChain**: Used for natural language processing and managing the flow of interactions.
- **Streamlit**: To build the web interface.
- **Ollama**: Running Mistral LLM locally to power the AI tutor.
- **SQLite**: For storing memory and learned concepts.
